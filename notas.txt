Puedo usar la librería ICal para pasar de un evento a un .ical

Campos obligatorios:
* UID: Es un identificador único para el evento. Debe ser único dentro del calendario.
* DTSTAMP: Fecha y hora en que se creó el evento, expresada en formato UTC (Tiempo Universal Coordinado).
* DTSTART y DTEND: Definen el inicio y el final del evento, respectivamente. Se deben expresar en formato UTC, pero si se trata de un evento de todo el día, solo se usa la fecha, sin la hora (e.g., DTSTART;VALUE=DATE:20230820).
* SUMMARY: Título o resumen del evento.

Ejemplo:
let event = emptyEvent
                { eventUID = Just "uid1@example.com"
                , eventDTStamp = Just currentTime
                , eventStartDateTime = Just startTime
                , eventEndDateTime = Just endTime
                , eventSummary = Just "Reunión de Proyecto"
                }
  
Si quiero agregar categorias lo hago con el siguiente Campos
                eventCategories = Just ["Trabajo"]


-------- Exportar ICal ----------
exportToICal :: FilePath -> [Event] -> IO ()
exportToICal file events = do
  let icalHeader = "BEGIN:VCALENDAR\nVERSION:2.0\nPRODID:-//Your Company//NONSGML Event//EN\n"
      icalFooter = "END:VCALENDAR\n"
      icalEvents = concatMap renderICalEvent events
  writeFile file (icalHeader ++ icalEvents ++ icalFooter)

renderICalEvent :: Event -> String
renderICalEvent event =
  let startDate = formatTime defaultTimeLocale "%Y%m%dT%H%M%S" (eventStartTime event)
      endDate = formatTime defaultTimeLocale "%Y%m%dT%H%M%S" (eventEndTime event)
      description = eventDescription event
      uid = eventUid event
      category = eventCategory event
  in unlines
     [ "BEGIN:VEVENT"
     , "UID:" ++ uid
     , "DTSTART:" ++ startDate
     , "DTEND:" ++ endDate
     , "SUMMARY:" ++ description
     , "CATEGORY:" ++ category
     , "END:VEVENT"
     ]
